== Description ==
Sync your new blog to wbto platform automatically
自动把你的新文章同步到微博通，支持标题+内容截断+抓取图像+更新不同步.

== Installation ==

To install: 

1. 直接在后台安装插件→上传→选择wbto.zip即可
